 API de Carros com Quarkus

Uma API para gerenciar carros, modelos e fabricantes.

 Funcionalidades

- Cadastro de carros com informações detalhadas (modelo, fabricante e ano).
- Gerenciamento de modelos de carros.
- Cadastro e consulta de fabricantes.

 Requisitos

- Java (versão 11 ou superior)
- Quarkus (versão mais recente recomendada)
- Banco de dados (PostgreSQL, MySQL ou outro suportado pelo Quarkus)
- Maven ou Gradle para gerenciamento de dependências
